# Vila Fi - Web Environment Hub

Um hub web interativo onde os usuários podem montar, personalizar e imergir em seus próprios ambientes de trabalho ou estudos. O projeto foca em fornecer rádios de lo-fi acompanhadas de um mixer de sons ambientes, com o grande diferencial de um sistema de vídeos de fundo que reage dinamicamente ao ambiente sonoro escolhido.

## 🎧 Principais Funcionalidades

- **Rádios Lo-Fi**: Escolha e alterne entre diferentes estações de rádio para o áudio principal.
- **Mixer de Sons Ambientes**: Adicione sons customizáveis como chuva, teclado mecânico, ruído branco, lareira, etc. e ajuste seus volumes independentemente.
- **Backgrounds Dinâmicos e Interativos**: O vídeo de fundo muda ou transita suavemente com base nas suas escolhas no mixer. (Ex: ativar o som de chuva muda a cena da cidade de ensolarada para chuvosa).

## 🚀 Como rodar o projeto localmente

1. **Clone o repositório**
   ```bash
   git clone https://github.com/josevilanir/vila_fi.git
   ```

2. **Instale as dependências**
   Na pasta do projeto, execute:
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento**
   ```bash
   npm run dev
   ```

4. **Acesse**
   Abra o seu navegador e acesse [http://localhost:3000](http://localhost:3000).
